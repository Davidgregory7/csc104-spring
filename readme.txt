U.S. Constitution - Styling Assignment

Description:
This project contains an excerpt from Article I of the U.S. Constitution styled with custom CSS. The objective is to create an engaging and visually appealing web page.

HTML File:
constitution.html

CSS File:
style.css

Styling Features:
- Modern typography using Georgia font.
- A responsive layout with readable text.
- Enhanced visuals with background colors, shadows, and rounded corners.
- A sticky footer and a styled header to frame the content effectively.
- Media queries to ensure compatibility across devices.

Validation:
Both the HTML and CSS files have been validated using:
- HTML Validator: https://validator.w3.org/
- CSS Validator: https://jigsaw.w3.org/css-validator/
Screenshots of validation results are attached to the submission.
